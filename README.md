# deal-forge

**deal-forge** is a web application to track deals on Steam. It is designed to run with a self-hosted PostgreSQL database and is intended as a hobby / personal project.  

## Features

- Periodically fetches and tracks Steam game deals  
- Stores deal history and metadata in PostgreSQL for persistence  
- Simple web interface for browsing current and historical deals  
- Easy self-hosting on your own infrastructure  

## Limitations & Known Issues

- This is a hobby project; not production-hardened   
- No guarantee of long-term support  

Feel free to treat it as-is or adapt for your own needs.
